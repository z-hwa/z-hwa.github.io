# Function

# main:

  • 匯入腳色模型 -> 匯入環境配佈 -> 切換腳色模型控制台 -> 匯入動作 -> 匯入臉部表情 -> 拖入音樂文件 -> 添加鏡頭配佈 -> 添加特效 -> 輸出

# Basic (操作完記得要 rigester):

# Interpolation curve:

用於調整動作之間的速度快慢
向上彎：減速過度、向下彎：加速過度
可套用不同的模型曲線，進一步調整

# model manipulation:

當前控制台對應模型修改

# camera/bone manipulation:

  1. 鏡頭模型中可調整鏡頭角度
    可選擇鏡頭鎖定的模型骨骼
  2. 腳色模型中可調整骨骼，編排動作

# light manipulation/facial manipulation:

  1. 鏡頭模型中可調整光線
  2. 腳色模型中可調整臉部表情

# accessory manipulation:

用於操作 MME 特效

# view:

切換視角 (滑鼠滾輪縮放、右鍵拖動)

# play:

播放動作

# Effect (支援 direct9，如為更高版本，下載 DirectX End-User Runtime Web):


可在該表中，調整每個模型使用的效果，或是取消顯示特定效果

# 幀:


可調整、察看每個模型在每一幀的狀況
輸出影片前，需先知道腳色模型動作的最後一幀，以進行輸出
